# sql-online-challenge

Unstyled SQL GUI

### Installing Locally
```java
git clone https://github.com/TheExpandingPolytope/sql-online-challenge.git
cd sql-online-challenge
npm install
```
in you .env file
```java
MONGO_URI=<YOUR MONGO DB STRING>
```

### Built With
ExpressJS
Passport
MongoDB
sqlite3
